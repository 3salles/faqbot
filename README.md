## Instalação
### Pré Requisitos
Git, node.js, npm

### Instalar Dependências
```bash

# Clone o repositório
$ git clone 

# Abra a pasta do servidor
$ cd chatbot

# Instale as dependências
$ yarn start

# Acesse a porta 
http://localhost:3333/
```
